# COWEL Language Support for VSCode

Syntax highlighting and language support for [Compact Web Language](https://cowel.org).

## Features

Currently, the plugin supports:

- syntax highlighting for COWEL markup files (`.cow`)
- support for directives, literals, comments, and escape sequences
- bnracket matching and auto-closing
- folding support for structured content

## Language Features

COWEL is a markup language with TeX-like syntax designed for generating HTML documents, particularly for 
technical proposals and C++ standards documents.
