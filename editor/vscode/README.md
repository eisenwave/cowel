# COWEL Language Support for VSCode

Syntax highlighting and language support for
[COWEL (Compact Web Language)](https://cowel.org).

## Features

Currently, the plugin supports:

- syntax highlighting for COWEL markup files (`.cow` and `.cowel`)
  - support for directives, literals, comments, and escape sequences
- bracket matching and auto-closing
- folding support for structured content
- minimal diagnostics backed by the COWEL WASM build

## Language Features

COWEL is a markup language with TeX-like syntax designed for generating HTML documents,
particularly for technical proposals and C++ standards documents.
